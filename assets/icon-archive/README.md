# ホーム画面アイコンの旧バージョン保管庫

現行アイコンのマスターは `docs/icon.svg`（雷主役＋右下に小さな領収書のブランド共通構図）。
ここには過去のデザインをSVG＋512px PNGで残す。復活させる場合はSVGを `docs/icon.svg` に
コピーし、Playwrightで192/512のPNGを再生成する（手順は hub の docs/BRAND_ASSETS.md 参照）。

| バージョン | ファイル | 構図 | 使用期間 |
|---|---|---|---|
| v1 | `icon-v1-receipt-behind.*` | 領収書を大きく背面に敷き、左傾き23.4°。雷は中央 | 2026-07-08 〜 07-16 |
| v2 | `icon-v2-receipt-mirrored-large.*` | v1の領収書を鏡写し（右傾き）にして右下へ寄せた大判構図 | 2026-07-16（短期） |

v2→現行への変更理由: 他の紫電アプリ（Dashboard・LifeLogger）は「雷が主役、モチーフは
タイル約1/3の右下ワンポイント」の構図のため、それに合わせた。大判のモチーフは
小サイズ表示で雷と絡んで潰れる、という教訓も hub の BRAND_ASSETS.md に記録済み。
