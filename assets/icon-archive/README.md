# ホーム画面アイコンのバージョン保管庫

現行アイコンのマスターは `docs/icon.svg`。**現行はv1**（領収書を大きく背面に敷いた初代。
他アプリと構図は異なるが「こちらのほうがかっこいい」というオーナーの好みで復活・確定）。
ここには歴代デザインをSVG＋512px PNGで残す。切り替える場合はSVGを `docs/icon.svg` に
コピーし、Playwrightで192/512のPNGを再生成する（手順は hub の docs/BRAND_ASSETS.md 参照）。

| v2 | `icon-v2-receipt-mirrored-large.*` | v1の領収書を鏡写し（右傾き）にして右下へ寄せた大判構図 | 2026-07-16（短期） |
